Actions

On the right side you can see the dynamically created animated drop-down list. It uses Native JavaScript and CSS and can be used anywhere in HTML document as a separate element.

When creating, you need to pass an object with four parameters:
    selector - string with CSS selector of parent node;
    theme - string with list title;
    url - string with resource address for download list values;
    onSelect - function for list interaction. Must be able:
        - log the received parameter;
        - respond to a click on the active button and start the
          corresponding method;
        - use the result returned by the method;
        - return boolean processing confirmation.

The created select itself controls the activation of the buttons and the use of its own methods. The expanded list can be scrolled using the keys Arrow Up/Down. Internal processes are encapsulated, from the outside it is possible to use only permitted methods, the change of which does not affect the select's work.
